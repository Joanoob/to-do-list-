# Todo-List Applikasjon

Dette er en enkel todo-list applikasjon utviklet i HTML CSS JavaScript

## Funksjonalitet

- Legg til oppgaver i todo-listen.
- Merk oppgaver som fullførte.
- Slett oppgaver du ikke lenger trenger.
